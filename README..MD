Project Title

SDG 3 – Good Health & Well-Being Information Website

Description

This project is a responsive informational website based on UN Sustainable Development Goal 3 (Good Health & Well-Being).
The website aims to raise awareness about global health issues, including HIV/AIDS, mental health, child health, vaccination, and health education, while presenting solutions, resources, and ways for users to get involved.

The website is designed to work on both desktop and mobile devices, featuring a responsive navigation menu and an image gallery slider on the homepage.

Technologies Used

HTML5 – Structure and content

CSS3 – Styling, layout, and responsive design

JavaScript – Interactive features

Swiper.js – Image slider for the homepage gallery

Features

Responsive navigation bar with hamburger menu for mobile view

Swiper image gallery on the homepage

Photo stories section using CSS Grid

Informational sections:

About SDG 3

Key Health Issues

Solutions & Actions

Resources

Contact and Get Involved forms

Mobile-friendly layout using media queries

How to Run the Project

Download or unzip the project folder

Make sure all files (index.html, index.css, script.js, images) are in the same directory

Open index.html in any modern web browser (Chrome, Edge, Firefox)


External Libraries

Swiper.js (via CDN)
Used for the homepage image slider.
